# 66bit employees

Сайт: http://194.87.29.195

Веб-приложение для просмотра списка сотрудников компании, разработанное в рамках тестового задания для попадания на практику в компанию **66bit** на направление **Frontend-разработчик**.

**Студент:** Муравьев Илья Германович

## Основной функционал

* **Фильтрация:** Поиск по имени, а также множественный выбор по категориям: должность, пол и технологический стек.
* **Синхронизация состояния:**
    * Все выбранные фильтры дублируются в query-параметрах URL и в LocalStorage.
    * При перезагрузке страницы фильтры автоматически восстанавливаются, обеспечивая сохранение результатов поиска.
* **Пагинация и бесконечный список:** Организована подгрузка данных с сервера, минимизирующая нагрузку на клиентскую часть.
* **Система смены тем:**
    * Поддержка светлой и темной тем оформления.
* **Адаптивная верстка:** Корректная работа приложения на устройствах с различными размерами экранов.

## Технологический стек

* **React 18** (Vite)
* **Redux Toolkit**
